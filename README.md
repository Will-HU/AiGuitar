# AiGuitar
在本系統以前端 Flutter/Dart 為主要開發框架，搭配 camera、audioplayers、permission_handler 等套件進行相機與音訊控制，並透過 EventChannel 實現即時麥克風串流；後端採用 Flask (Python) 建立 REST API，結合 YOLO 與 PyTorch 進行視覺和弦辨識，以及 librosa/torchaudio 的音訊分析，並透過 OpenCV 與 NumPy 完成前處理。系統以 Firebase 提供會員登入、資料儲存與雲端服務，開發過程中使用 Visual Studio Code 與 Android Studio 進行程式設計與除錯
